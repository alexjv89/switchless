# Installation

Install Switchless, a carefully crafted library for building Cashflowy's products.

```bash
npm install switchless
```



## Detailed Installation

For a more controlled installation process:

1. Clear npm cache (Recommended if you're having issues):
   ```bash
   npm cache clean --force
   ```

2. Install a specific version:
   ```bash
   npm install switchless@x.x.x
   ```
   Replace `x.x.x` with the desired version number.

## Verifying Installation

After installation, you can verify it by checking your `package.json` file or running:

```bash
npm list switchless
```
## Troubleshooting

If you encounter any issues during installation [open an issue](https://github.com/alexjv89/switchless) on our GitHub repository.
