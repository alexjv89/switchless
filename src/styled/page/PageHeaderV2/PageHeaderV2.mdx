## Overview

The `PageHeaderV2` component is designed to provide a consistent and customizable header for your pages. It includes support for breadcrumbs, a main title, and optional action buttons.

## Basic Usage




```jsx
import { PageHeaderV2 } from 'switchless';

<PageHeaderV2 header="Dashboard"/>
```

## Props

The `PageHeaderV2` component accepts the following props:

- `header`: (required) The main title of the page. Can be:
  - A string for a simple header
  - An object with `part1` and `part2` properties for a two-part header with different styling
- `headerLevel`: (optional) The heading level for the main title. Defaults to 'h3'.
- `breadcrumbs`: (optional) An array of breadcrumb objects with `text` and optional `href` properties.
- `RightButtons`: (optional) A component or function that returns action buttons for the header.

## Header as an Object
You can pass an object to the `header` prop to create a two-part or three-part header.
The object can have `part1`, `part2`, and `part3` properties. Each with a different styling hardcoded in the component.

```jsx
<PageHeaderV2 
header={{ part1: 'Three ', part2: 'Part', part3: 'Header' }} 
/>

```
