# Logo Component

<Subtitle>
 Logo component for Cashflowy Applications.
</Subtitle>

The Logo component displays a logo with optional customization options.


```jsx
import { Logo } from 'switchless';

```
## Props

The Logo component accepts the following props:

- `offering`: (optional) Custom text to display alongside the logo
- `href`: (optional) Custom link destination
