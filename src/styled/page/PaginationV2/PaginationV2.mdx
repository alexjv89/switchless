# Pagination Component

<Subtitle>
  A flexible pagination component for navigating through multiple pages.
</Subtitle>



```jsx
import { PaginationV2 } from 'switchless';

<PaginationV2 pageCount={10} />
```

## Props

The Pagination component accepts the following props:

- `pageCount`: (required) The total number of pages.
- `disabled`: (optional) If true, disables all pagination controls. Defaults to false.
